## 3. 能力依赖

- App 版本
  - 智能生活 5.13.0 及以上版本
- TTT 依赖
  - BaseKit: 3.11.3
  - MiniKit: 3.9.8
  - DeviceKit: 4.5.1
  - BizKit: 3.0.8
  - HomeKit: 3.4.0
  - baseversion: 2.19.2
- 组件依赖
  - 暂无
- 功能页依赖
  - 设备详情功能页：settings => 'tycryc71qaug8at6yt'
