  - Smart Life 5.13.0 and above

  - MiniKit: 3.9.8
  - DeviceKit: 4.5.1
  - BizKit: 3.0.8
  - HomeKit: 3.4.0
  - baseversion: 2.19.2
- Component dependency
  - None
- Function page dependency
  - Device Detail Functional Page: settings => 'tycryc71qaug8at6yt'
