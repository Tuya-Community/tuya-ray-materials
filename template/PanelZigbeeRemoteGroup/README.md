## 3. Capability dependency

- App version
  - Smart Life 5.1.0 and above
- TTT dependency
  - BaseKit: 3.0.0
  - MiniKit: 3.0.1
  - DeviceKit: 3.0.2
  - BizKit: 3.0.6
  - HomeKit: 3.1.3
  - baseversion: 2.13.3
- Component dependency
  - None
- Function page dependency
  - Device Detail Functional Page: settings => 'tycryc71qaug8at6yt'
