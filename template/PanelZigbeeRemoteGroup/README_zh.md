## 3. 能力依赖

- App 版本
  - 智能生活 5.1.0 及以上版本
- TTT 依赖
  - BaseKit: 3.0.0
  - MiniKit: 3.0.1
  - DeviceKit: 3.0.2
  - BizKit: 3.0.6
  - HomeKit: 3.1.3
  - baseversion: 2.13.3
- 组件依赖
  - 暂无
- 功能页依赖
  - 设备详情功能页：settings => 'tycryc71qaug8at6yt'
