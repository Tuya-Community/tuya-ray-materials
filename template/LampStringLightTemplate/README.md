## Project Name: Lighting Colorful Light Strip Template

## 1. Usage Notice

Before using this template for development, you need to have a basic understanding of the Ray framework. It is recommended to first read the [Ray Development Documentation](https://developer.tuya.com/cn/miniapp/develop/ray/guide/overview)

## 2. Quick Start

- [Create Product](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panel-lamp-strip/index.html#2)
- [Create Project and Import Project Code in IDE](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panel-lamp-strip/index.html#3)

## 3. Capability Dependencies

- App Version

  - Smart Life 5.0.0 and above

- TTT Dependencies

  - BaseKit: 3.3.13
  - MiniKit: 3.1.0
  - DeviceKit: 3.4.0
  - BizKit: 3.3.1
  - baseversion: 2.10.1

- Function Page Dependencies
  - Timer Countdown Function Page: LampScheduleSetFunction

## 4. Panel Features

- Colorful Light String Control:
  - Support multiple modes
- Scenes:
  - 80+ built-in scene modes
- Music Rhythm:
  - Support local and cloud music rhythm
- DIY Scene Editing
- Multi-segment Light Bead Control:
  - Can select light bead segments for independent control
- Multi-protocol Device Support:
  - Support WiFi + Bluetooth
- Group Function:
  - Support group function

## 5. Feature Implementation

- **Light Control**: Color/White light mode switching
- **Scene Management**: Built-in scenes + custom scenes
- **Music Sync**: Local/cloud music rhythm
- **Edit Function**: DIY color editor

### Special Implementation

- **Smear Function**: Support paint bucket and single-point smearing
- **Segment Control**: Can select light bead segments for independent control
- **Data Sync**: Remote control operation and panel status synchronization
- **Performance Optimization**: Rendering optimization for large amount of light bead data
