## 项目名称：AI 图像优化模板

## 1. 使用须知

使用该模板开发前，需要对 Ray 框架有基本的了解，建议先查阅 [Ray 开发文档](https://developer.tuya.com/cn/miniapp/develop/ray/guide/overview)

## 2. 快速上手

[使用 Ray 开发万能面板](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panelmore-guide/index.html#0)

## 3. 能力依赖

- App 版本
  - 智能生活 6.9.0 及以上版本
- TTT 依赖
  - BaseKit: 3.0.6
  - MiniKit: 3.0.0
  - DeviceKit: 3.0.0
  - BizKit: 3.0.1
  - baseversion: 2.10.29
  - HomeKit: 3.0.2
  - AIKit: 1.4.3
- 组件依赖
  - @ray-js/smart-ui

## 4. 面板功能

- 图像清晰度增强处理
- 图像畸变校正

## 5. 功能实现

- 参考 [AI 图像增强处理](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panel-ai-picture-enhance/index.html#0)
