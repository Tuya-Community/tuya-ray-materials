## Usage Notice

Before developing with this template, you need to have a basic understanding of the Ray framework. It is recommended to first consult the [Ray Development Documentation](https://developer.tuya.com/cn/ray).

## 3. Capability Dependencies

- **App Version Requirements:**

  - Tuya App, Smart Life App v6.6.0 and above

- **Toolkit Dependencies:**

  - BaseKit: v3.0.6
  - MiniKit: v3.0.1
  - DeviceKit: v4.14.0
  - BizKit: v4.2.0
  - WearKit: v1.1.6
  - baseversion: v2.27.0

- **Component Dependencies:**
  - @ray-js/ray: "1.7.14",

## 4. Panel Features

- **Recording File Operations:**

  - Delete recording files
  - Recording history list
  - File renaming

- **Transcription:**
  - Generate summary
  - Play recording files

- **Real-time Recording:**
  - Real-time recording
  - Real-time transcription
  - Real-time translation
  - Face-to-face translation
  - Call recording
  - Meeting recording
