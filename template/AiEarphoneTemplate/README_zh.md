## 项目名称：AI耳机模板

## 使用须知

使用该模板开发前， 需要对 Ray 框架有基本的了解，建议先查阅[Ray 开发文档](https://developer.tuya.com/cn/ray)

## 2. 快速上手

- [创建产品](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panel-ai-audio-transcription-and-summary/index.html#1)
- [创建项目并在 IDE 中导入项目代码](https://developer.tuya.com/cn/miniapp-codelabs/codelabs/panel-ai-audio-transcription-and-summary/index.html#2)

## 3. 能力依赖

- **App 版本要求：**

  - 涂鸦 App、智能生活 App v6.6.0 及以上版本

- **工具包依赖：**

  - BaseKit: v3.0.6
  - MiniKit: v3.0.1
  - DeviceKit: v4.14.0
  - BizKit: v4.2.0
  - WearKit v1.1.6
  - baseversion: v2.27.0

- **组件依赖：**
  - @ray-js/ray: "1.7.14",

## 4. 面板功能

- **录音文件相关操作：**

  - 录音文件删除
  - 录音记录列表
  - 文件重命名

- **转录：**
  - 生成总结
  - 播放录音文件

- **实时录音：**
  - 实时录音
  - 实时转写
  - 实时翻译
  - 面对面翻译
  - 电话录音
  - 会议录音
